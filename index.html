Amherst Campus Locations Map

This interactive map highlights five key UMass Amherst landmarks—Campus Pond, Minuteman Statue, Du Bois Library Classroom, Morrill Geosciences, and Holdsworth Environmental Conservation. Built using the Google Maps JavaScript API and AdvancedMarkerElement, it auto‑zooms to fit all markers on load, offers click‑activated info pop‑ups, and includes a Dark / Light toggle and “📍 Me” geolocation button. Improvements since the initial submission reflect lecture and demo learnings on fitBounds, theming, and interactive UI components, ensuring enhanced usability and mobile responsiveness.

What’s new since the initial map submission?
  • Swapped basic Marker for AdvancedMarkerElement
  • Wrapped markers in LatLngBounds with map.fitBounds(bounds)
  • Added InfoWindow pop‑ups on marker click
  • Built Dark / Light toggle using CSS variables
  • Implemented “📍 Me” geolocation button
  • Refactored CSS for viewport‑filling responsive map container
  • Added alt text and external link best‑practices

Healthcare Accessibility Web App

This ArcGIS Online web app visualizes hospital coverage zones, 10‑minute drive‑time areas, counts of health centers per town, nearest hospital per census block, and underserved regions identified via overlay analysis. Interactive filters allow users to toggle between buffer and drive‑time analyses, adjust symbology with a color‑blind friendly palette, and search addresses using the Nearby widget. Enhancements include redesigned pop‑ups, mobile‐responsive layouts, and embedded statistical summaries for data‐driven decision‐making.

What’s new since the initial ArcGIS app submission?
  • Re‑styled symbology with a color‑blind safe palette
  • Optimized pop‑up content to remove scrollbars
  • Added Nearby widget for address search
  • Enabled mobile‐responsive sidebar layout
  • Included summary charts in pop‑ups via Arcade expressions
  • Embedded responsive filter sliders

Survey123 Feedback Collection & Dashboard

This Survey123 solution captures community feedback on local GIS workshops, allowing participants to submit ratings, comments, and photos. The live results dashboard displays dynamic bar charts, pie graphs, and a filtered table of responses embedded within Experience Builder. Users can filter by workshop date, attendee role, and rating. The setup demonstrates real-time data collection and interactive visualization to inform future training sessions.

What’s new since the initial survey form submission?
  • Enabled anonymous submissions and photo attachments
  • Designed real-time dashboard with dynamic chart widgets
  • Embedded dashboard into Experience Builder page
  • Added filter controls for date range and attendee role
  • Custom themed page layout following branding guidelines
  • Tested mobile form view and dashboard responsiveness
